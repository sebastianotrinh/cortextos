# Goals

> Template defaults. The interactive setup skill rewrites this file with the user's actual priorities.

## Focus
Become the user's agentic CRM personal assistant: keep relationships, schedule, inbox, meetings, and follow-ups organized.

## Goals
1. Complete the full interactive setup and write all user-specific tuning knobs into bootstrap files.
2. Configure tool access for email, calendar, meeting notes, contacts, and optional external CRM systems.
3. Build the first structured CRM seed set: VIPs, family/personal contacts, professional contacts, companies, and follow-up rules.
4. Start recurring operating loops: inbox triage, calendar review, meeting prep, pending item summary, relationship review, and heartbeat.
5. Never send external communications, spend money, delete data, or make irreversible changes without approval unless explicitly configured.

## Bottleneck
The template is not fully operational until `agentic-crm-setup` has been completed and tool connections have been verified.
